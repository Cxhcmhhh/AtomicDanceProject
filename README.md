# AtomicDanceProject

Project page for **Music-to-Dance Generation via Atomic Movements**.

This static site hosts the AtomicDance project page, with video and figure
areas kept as clear placeholders until final media assets are ready.

## Local Preview

Open `index.html` directly in a browser, or serve the directory with any static
file server:

```bash
python -m http.server 8000
```

## Assets To Replace

- `static/pdfs/paper.pdf`: current AtomicDance paper PDF.
- Teaser and result placeholders in `index.html`.
- Pipeline placeholder in `index.html`.
- `static/images/social_preview.png` when a social preview image is available.
